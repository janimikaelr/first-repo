# Sales Data Analysis

This project analyzes retail store sales data to examine regional performance and operational efficiency.

The dataset includes 2,485 stores, of which 2,199 were analyzed after data cleaning. Stores were segmented by region (East, North, South, West) and by size (Small, Medium, Large).

Main findings:
- Revenue is concentrated in the North and West regions.
- Larger stores generate higher average sales.
- Significant performance differences exist within similar store categories.
- East and South regions show lower average sales and customer footfall.

The repository includes the full analysis notebook, dataset, and written business report.
