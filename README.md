# Sales Data Analysis

This project analyzes retail store sales data to evaluate revenue distribution, regional performance differences, and operational efficiency.

## Project Overview
The dataset originally contained 2,485 stores. After removing missing values, 2,199 stores were included in the analysis. The stores were segmented by region (East, North, South, West) and by size (Small, Medium, Large).

The objective was to:
- Understand revenue concentration across regions
- Compare performance between store sizes
- Identify performance gaps within the same region and category
- Provide data-driven managerial recommendations

## Key Insights
- Revenue is heavily concentrated in the North and West regions.
- Larger store size correlates with higher average sales.
- Significant performance differences exist between stores operating under similar conditions.
- East and South regions show lower average customer footfall and sales.

## Methods & Tools
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Performance score development (Sales per Employee + Average Customer Sales)
- Python (Pandas, Matplotlib)

The repository includes the Jupyter Notebook, dataset, and the final written report.
